Single Page Applications

Single Page Applications

Single Page Applications（SPA, 单页应用程序）是指在浏览器中运行的应用，所谓单页也就是页面始终都只有一个页面，它们在使用期间不会重新加载页面。

更准确的说就是只通过单个URL就能完整访问的应用程序。

React：

  • 是一个 SPA 概念的的技术实现；
  • 是一个渐进式的前端技术栈；
