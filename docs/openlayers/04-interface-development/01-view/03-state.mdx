# 组件状态

## 大纲

- 状态
  - 声明
    - 初始值
      - 默认值来源
      - 初始化函数-避免重复创建初始状态
  - 读取
  - 更新
    - 派生状态/计算
    - 更新后立即获取
      - setState(() => {})
      - 保存变量中
    - 状态变更立即更新 DOM
    - 更新状态中的对象和数组
      - 不可变数据，替换
      - 扩展符 ...
    - 性能优化
      - 避免重复创建初始状态
      - 使用 key 重置状态
      - 存储前一次渲染的信息
  - 监听
    - 异步监听
    - 同步监听
    - 深度监听和立即执行
    - 监听依赖自动收集
