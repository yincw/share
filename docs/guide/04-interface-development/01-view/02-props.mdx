## 属性绑定

JSX 是一种模板语言的最小实现，因为它允许你通过 JavaScript 来组织数据和逻辑。

```tsx showLineNumbers
export default function Profile() {
  const other = {
    name: 'Gregorio Y. Zara',
    theme: {
      backgroundColor: 'black',
      color: 'pink'
    }
  };

  return (
    // 属性绑定
    // 使用“引号”传递字符串
    // 使用“大括号”传递 JavaScript 变量，大括号内的任何 JavaScript 表达式都能正常运行，包括函数
    // 使用“双大括号”传递 JavaScript 对象
    // highlight-next-line
    <Avatar person="haha" size={50} other={other}  />
  )
}
```

## Children 属性

当你将内容嵌套在 JSX 标签中时，父组件将在名为 `children` 的 prop 中接收到该内容。

```tsx showLineNumbers
function Card({ children }) {
  return (
    <div className="card">
      {/* children 属性读取 */}
      // highlight-next-line
      {children}
    </div>
  );
}

export default function Profile() {
  return (
    <Card>
      {/* children 属性 */}
      <Avatar
        size={100}
        person={{ 
          name: 'Katsuko Saruhashi',
          imageId: 'YfeOqp2'
        }}
      />
    </Card>
  );
}
```

:::tip

## Render Props

除了为每一个子项生成 JSX，你还可以传递一个返回值类型是 JSX 的函数，并且在必要的时候调用这个函数。

```tsx showLineNumbers
import { useState } from 'react';

export default function TabSwitcher({ tabIds, getHeader, renderContent }) {
  const [selectedId, setSelectedId] = useState(tabIds[0]);
  return (
    <>
      {tabIds.map((tabId) => (
        <button
          key={tabId}
          onClick={() => setSelectedId(tabId)}
        >
          {/* 渲染函数读取 */}
          // highlight-next-line
          {getHeader(tabId)}
        </button>
      ))}
      <hr />
      <div key={selectedId}>
        {/* 渲染函数读取 */}
        // highlight-next-line
        <h3>{getHeader(selectedId)}</h3>
        // highlight-next-line
        {renderContent(selectedId)}
      </div>
    </>
  );
}

export default function App() {
  return (
    <TabSwitcher
      tabIds={['first', 'second', 'third']}
      {/* Render Props 渲染属性 */}
      getHeader={tabId => {
        return tabId[0].toUpperCase() + tabId.slice(1);
      }}
      {/* Render Props 渲染属性 */}
      renderContent={tabId => {
        return <p>This is the {tabId} item.</p>;
      }}
    />
  );
}
```

渲染属性是函数，所以你可以向它们传递参数。

- https://zh-hans.react.dev/reference/react/Children#calling-a-render-prop-to-customize-rendering
