# 运算符和操作符

- 一元操作符
    - `++` 递增
    - `--` 递减
    - `+` 一元加（隐式类型转换）
    - `-` 一元减（隐式类型转换，负数）
- 位操作符
    - `~` 按位非（NOT）
    - `&` 按位与（AND）
    - `|` 按位或（OR）
    - `^` 按位异或（XOR）
    - `<<` 左移
    - `>>` 有符号右移
    - `>>>` 无符号右移
- 布尔操作符
    - `!` 逻辑非（NOT）
    - `&&` 逻辑与（AND）
    - `||` 逻辑或（OR）
- 乘性操作符
    - `*` 乘法
    - `/` 除法
    - `%` 求模（余数）
- 加性操作符
    - `+` 加法
    - `-` 减法
- 关系操作符
    - `<` 小于
    - `>` 大于
    - `<=` 小于等于
    - `>=` 大于等于
- 相等操作符
    - `==` 相等
    - `!=` 不相等
    - `===` 全等
    - `!==` 不全等
- 条件操作符
    - `?:` 三元运算符
- 赋值操作符
    - `=` 赋值
    - `+=` 加赋值
    - `-=` 减赋值
    - `*=` 乘赋值
    - `/=` 除赋值
    - `%=` 模赋值
    - `<<=` 左移赋值
    - `>>=` 有符号右移赋值
    - `>>>=` 无符号右移赋值
    - `**=` 指数赋值
- 逗号操作符
    - `,`
- 指数操作符
    - `**`
- 扩展操作符
    - `...` 展开（扩展）
- 传导操作符
    - `?.` 存在
- 其他操作符
    - 变量
        - var
        - let
        - const
    - 类型和实例
        - typeof
        - instanceof
        - new
    - 对象
        - delete
    - 语句
        - return
        - break
        - in
        - of
    - 错误
        - throw
    - 异步编程
        - await
        - yield
    - 模块
        - import
        - export
- 隐式类型转换
