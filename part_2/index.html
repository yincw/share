Props 和 State

- props state
能否从父组件获取初始化值？ Yes Yes
能否在组件内设置默认值？ Yes Yes
能否设置子组件的初始值？ Yes Yes
能否被父组件改变？ Yes No
能否在组件内改变？ No Yes
能否在子组件中改变？ Yes No
  • 注意：props 和 state 从父组件获取的初始值都会被在组件内设置的默认值覆盖。
  • Instance Properties

React Props

  • props 就是 properties（属性） 的缩写，你可以使用它把任意类型的数据传递给组件。
  • 可以通过 this.props 访问 props，但绝对不能通过这种方式修改它。一个组件绝对不可以自己修改自己的 props。

React State

  • 每一个React 组件都拥有自己的 state，state 与 props 的区别在于前者只存在于组件的内部。
  • state 可以通过 setState 来修改，也可以使用 getInitialState 方法提供的一组默认值。只要 setState 被调用，render 就会被调用。
  • 千万不能直接修改 this.state，永远记得要通过 this.setState 方法修改。
